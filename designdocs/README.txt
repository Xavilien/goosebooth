# Description
> A short description of the project's name and what it does. This may be taken from your design docs.
Players navigate a 2D world as a mischievous goose. The gameplay involves interacting with the surroundings by shaking trees, consuming apples, and getting into trouble with humans. Have fun teasing a child and leading a goose-hater on a wild goose chase! But beware: Don't get caught...

# How to run the project
Run /scripts/main.py
Click on Goose Guidebook at the top left corner for further instructions

* No libraries need to be installed
* No shortcut commands (sorry) (but row/col of human characters are printed in console so you can estimte where you'd find them. Goose starts off in the midlde of the grid.)
* No fonts need to be installed
